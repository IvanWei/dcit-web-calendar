# DCIT-Calendar

## Description

DCIT-Calendar is calendar version with DCIT. This data gets [Google sheet](https://go.ivanwei.co/dcit-data-google-sheet) via [dcit-gsa](https://github.com/IvanWei/dcit-gas).

